| **项目摘要** |
| :---: |
| 本项目记录了一名高中生如何仅凭个人资金，独立从零开始设计、构建和编码一个由人工智能驱动的智能眼镜原型。“嵌入式人工智能视觉眼镜”(EGG) 是一款屡获殊荣的辅助设备，旨在通过解读环境来增强视障人士的独立性。它通过物体识别、文本转语音和实时翻译，将抽象的视觉概念转化为有形信息。|
